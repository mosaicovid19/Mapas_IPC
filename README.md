# Mapas_IPC
Gerador de mapas para espacialização do Índice de Proteção ao Covid-19

Para executar o gerador, é necessário abrir a pasta app com um visualizador (VS Code por exemplo), e abrir um terminal. 

No terminal, digite: python app.py

Ele irá utilizar os arquivos geojson de cada local que estão na pasta app\data. 
Ele também utilizará os dados do IPC para cada bairro, que estão na subpasta app\data\processed.
